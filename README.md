# chat-widget

### Start

##### 1) Run a web-socket server
...\chat-widget\server
```
npm run prod
```

##### 2) Run a manager server
...\chat-widget\manager
```
ng serve
```

##### 3) Run a widget server
...\chat-widget\client\express
```
npm run start
```

### Open apps

##### 1) Open a manager app
http://localhost:4200/

##### 2) Open a widget
...\chat-widget\client\index.html
